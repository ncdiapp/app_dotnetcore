
1. wj-combo-box default select the first item if selected-value is null. We can put selected-value = null in code behind to empty the selected filed.
	If required="true" (Default is true), no matter selected-value is null or not, the first field is always selected.

<wj-combo-box
	items-source="dictCriteriaOperatorDataSource['@criteria.SearcDCUID']" 
	display-member-path="Display" 
	@*selected-value-path="OperatorType"*@
	selected-value="dictDcuOperator['@criteria.SearcDCUID']"
	required="false"
	style="width: 100%;">
</wj-combo-box>
