Report Engine Reference

Token Mapping Guide

How data from stored procedures and REST APIs maps to {{tokens}} in your report template.

Stored Procedure SP
① Stored procedure
CREATE PROCEDURE sp_StyleReport @MainReferenceId INT, @MasterReferenceId INT = NULL AS BEGIN -- Result set 0 (1 row → scalar) SELECT StyleNumber, Brand, Season FROM Style WHERE StyleId = @MainReferenceId -- Result set 1 (rows → list) SELECT Code, Description, Colour FROM StyleFabric WHERE StyleId = @MainReferenceId -- Result set 2 (rows → list) SELECT ColourCode, ColourName FROM StyleColour WHERE StyleId = @MainReferenceId END
② Result sets → context keys
RS0 header — 1 row → scalar fields
StyleNumber, Brand, Season
RS1 header_rs1 — multiple rows → loop
Code, Description, Colour
RS2 header_rs2 — multiple rows → loop
ColourCode, ColourName
③ Available tokens
{{header.StyleNumber}} {{header.Brand}} {{header.Season}} {{#each header_rs1}} {{Code}} {{Description}} {{Colour}} {{#each header_rs2}} {{ColourCode}} {{ColourName}}
④ Template HTML
<h1>{{header.StyleNumber}}</h1> <p>{{header.Brand}} — {{header.Season}}</p> <table> {{#each header_rs1}} <tr> <td>{{Code}}</td> <td>{{Description}}</td> <td>{{Colour}}</td> </tr> {{/each}} </table> <h3>Colours</h3> {{#each header_rs2}} {{ColourCode}} — {{ColourName}} {{/each}}
REST API API
① API endpoint + sample JSON
Path: /webapi/Product/GetDetail  ·  Source name: product
{ "ProductCode": "P001", ← primitive → scalar "Description": "Cotton Shirt", "Brand": "CGS", "Fabrics": [ ← array → loop { "Code": "F001", "Name": "Cotton", "Pct": 100 } ], "Colours": [ ← array → loop { "Code": "BLK", "Name": "Black" }, { "Code": "WHT", "Name": "White" } ] }
② JSON properties → context keys
primitive product — scalar fields
ProductCode, Description, Brand
"Fabrics": […] product_Fabrics — loop
Code, Name, Pct
"Colours": […] product_Colours — loop
Code, Name
③ Available tokens
{{product.ProductCode}} {{product.Description}} {{product.Brand}} {{#each product_Fabrics}} {{Code}} {{Name}} {{Pct}} {{#each product_Colours}} {{Code}} {{Name}}
④ Template HTML
<h1>{{product.ProductCode}}</h1> <p>{{product.Description}} — {{product.Brand}}</p> <table> {{#each product_Fabrics}} <tr> <td>{{Code}}</td> <td>{{Name}}</td> <td>{{Pct}}%</td> </tr> {{/each}} </table> {{#each product_Colours}} {{Code}} — {{Name}} {{/each}}
Quick Reference — All Token Syntax
Scalar token Loop token Field inside loop
Token What it outputs SP source API source
{{header.StyleNumber}} Single field from a scalar row RS0 with 1 row Primitive property in JSON
{{#each header_rs1}} Loop — repeats inner HTML for every row RS1 (2nd result set) — not used for API
{{#each product_Fabrics}} Loop — array property "Fabrics" — not used for SP Array property named "Fabrics"
{{Code}} Field — only valid inside {{#each}} Column name in result set Key in each array item
{{/each}} Closes the loop block Same for both SP and API
{{#if header.Notes}} Shows inner HTML only when field is not empty/0/false Same for both SP and API
{{#each header}} Loop over RS0 when it returns multiple rows RS0 with many rows Root-level JSON array
{{#each header_rs2}} Loop over the 3rd result set RS2 (3rd result set) — not used for API
